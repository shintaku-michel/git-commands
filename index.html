Comandos úteis GIT

Direto ao ponto.

GIT WORKFLOW

  • WORKING DIRECTORY

    git init

  • STAGE AREA

    git add

  • LOCAL REPOSITORY

    git commit

COMANDOS

  • git init

    Inicia um WORKING DIRECTORY

  • git add .

    Adicionar todos arquivos WORKING DIRECTORY ao STAGE AREA

  • git add * .md

    Adicionar todos arquivos com extensão .md ao STAGE AREA

  • git log -n 5

    Lista os últimos 5 commits

  • git log --since 2023-01-25

    Lista os commits desde a data

  • git log --until 2023-01-25

    Lista os commits anteriores a data

  • git log --author Michel

    Lista os commits de um colaborador específico

  • git log --grep mobile

    Lista de commits com comentário específico

  • git status

    Verifica o estado do GIT WORKFLOW

  • git rm -- cached file-name to unstage

    Remove arquivo do STAGE AREA

  • git commit - m "mensagem"

    Incluir arquivo no LOCAL REPOSITORY

  • git restore nome-arquivo ou git reset HEAD nome-arquivo

    Descarta as alterações em um arquivo específico

  • git restore . ou git reset HEAD .

    Descarta todas as alterações realizadas

  • git diff .

    Verifica alteração no arquivo WORKING DIRECTORY

  • git diff --staged ou git diff --cached

    Verifica alteração nos arquivos no STAGE AREA

  • git rm nome-arquivo

    Remover arquivos do WORKING DIRECTORY

  • git mv nome-antigo novo-nome

    Renomear arquivo pelo git (renamed)

  • git mv nome-arquivo nome-pasta/nome-arquivo

    Mover arquivo para um novo diretório

  • git commit --amend -m "nova-mensagem"

    Corrigindo mensagem do último commit

  • git checkout -- ID_commit nome-arquivo

    Recupera um arquivo alterado errado

GIT MENSAGENS

  • Untracked files { use "git add file-name" to include in what will be committed }

    Arquivos não rastreados { use "git add nome-arquivo" para incluir este arquivo a ser comitado}

  • use "git restore file-name" to discard changes in working directory

    Use o comando "git restore file-name" para descartar mudanças no WORKING DIRECTORY

  • LOCAL REPOSITORY

    git commit
