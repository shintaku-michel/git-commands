Comandos úteis TERMINAL

Direto ao ponto.

TERMINAL

  • cd

    Mostra o diretório raiz do computador.

  • cd nome-da-pasta

    Troca de diretório.

  • cd ..

    Retorna para o diretório anterior.

  • ls

    Lista os arquivos e diretórios existentes.

  • mkdir nome-da-pasta

    Cria um novo diretório (pasta).

  • touch nome-do-arquivo

    Cria um arquivo (.txt, .htm, .css, .js ...).

  • vim nome-do-arquivo

    Edita um arquivo (.txt, .htm, .css, .js ...).

  • esc : wq

    Salva arquivo editado.

  • esc : q!

    Sair sem salvar o arquivo.
